# Корпоративные стандарты

## Обзор

Корпоративные стандарты определяют обязательные, рекомендуемые и устаревшие требования и практики, которые формируют ваш ИТ-ландшафт. Они обеспечивают согласованность, взаимодействие и соответствие во всей архитектуре вашей организации.

Модуль Корпоративные стандарты Turbo EA позволяет:

- **Определять стандарты** с категориями (безопасность, взаимодействие, данные, бизнес и т. д.)
- **Устанавливать уровни соответствия** (обязательный, рекомендуемый, устаревший)
- **Связывать стандарты с карточками** (приложения, компоненты ИТ и т. д.) и отслеживать соответствие
- **Проверять статус соответствия** с доказательствами и примечаниями для каждой пары карточка-стандарт
- **Ссылаться на внешние стандарты** (NORA, ISO, TOGAF и т. д.)

## Управление стандартами

### Просмотр доступных стандартов

Стандарты отображаются на вкладке **Управление** в модуле **GRC** (`/grc?tab=governance`).

Дополнительную информацию см. в [связанных темах](#связанные-темы).

## Лучшие практики

1. Начните с обязательных стандартов
2. Поэтапно связывайте карточки
3. Собирайте доказательства по мере продвижения
4. Четко документируйте исключения
5. Регулярно проверяйте статус соответствия
6. Интегрируйте с реестром рисков

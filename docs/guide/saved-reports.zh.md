# 已保存报告

Turbo EA 允许您**保存报告配置**，以便您可以快速返回特定视图，而无需每次重新配置筛选条件和轴。

## 保存报告

从任何报告页面（投资组合、能力地图、生命周期、依赖关系、成本、矩阵、数据质量或 EOL）：

1. 使用您期望的筛选条件、分组和轴选择配置报告
2. 点击报告工具栏中的**保存**按钮
3. 输入已保存报告的**名称**
4. 选择**可见性**：

| 可见性 | 谁可以看到 |
|--------|-----------|
| **私有** | 仅您自己 |
| **共享** | 您和您选择的特定用户 |
| **公开** | 平台的所有用户 |

对于共享报告，您可以向特定用户授予**编辑权限**，允许他们更新已保存的配置。

5. 点击**保存** —— 系统会自动从当前可视化捕获缩略图

## 已保存报告库

导航到**报告 > 已保存报告**浏览您有权访问的所有已保存报告。库以标签页组织显示缩略图预览：

- **我的报告** —— 您创建的报告
- **与我共享** —— 他人与您共享的报告
- **公开** —— 所有人可见的报告

### 操作

- **打开** —— 点击报告以加载其保存的配置
- **编辑** —— 更新名称、可见性或共享设置
- **复制** —— 使用新名称创建副本
- **删除** —— 移除已保存的报告（仅创建者或具有编辑权限的用户可以删除）

# Enterprise-standarder

## Oversigt

Enterprise-standarder definerer de obligatoriske, anbefalede og forældede krav og praksisser, der former dit IT-landskab. De etablerer konsistens, interoperabilitet og compliance på tværs af din organisations arkitektur.

Turbo EAs Enterprise-standarder-modul gør det muligt for dig:

- **Definere standarder** med kategorier (sikkerhed, interoperabilitet, data, forretning osv.)
- **Indstille compliance-niveauer** (obligatorisk, anbefalet, forældet)
- **Linke standarder til kort** (programmer, IT-komponenter osv.) og spore compliance
- **Revidere compliance-status** med bevis og noter for hvert kort-standard-par
- **Referere eksterne standarder** (NORA, ISO, TOGAF osv.)

# 仪表盘

仪表盘是登录后看到的第一个界面。它提供整个企业架构状态的**快速概览**。

![仪表盘 - 顶部视图](../assets/img/en/01_dashboard.png)

## 顶部导航栏

在屏幕顶部，您会看到包含以下元素的**主导航栏**：

- **Turbo EA**（logo）：点击可从任何页面返回仪表盘
- **仪表盘**：架构状态概览
- **清单**：所有卡片的完整列表
- **报告**：可视化和分析报告
- **BPM**：业务流程管理（如果已启用）
- **图表**：可视化架构图表编辑器
- **EA 交付**：架构项目管理
- **待办**：待处理任务和分配的问卷
- **搜索卡片**：带自动补全的快速搜索栏
- **+ 创建**：快速创建新卡片的按钮
- **通知铃铛**：系统提醒和[通知](notifications.md)
- **个人头像**：语言选择、主题切换、通知偏好和管理入口

## 摘要卡片

仪表盘的主要区域显示**摘要卡片**，包括：

- **卡片总数**：平台中注册的所有组件数量
- **按类型分布**：每种类型有多少元素（应用程序、组织、目标、能力等）
- **状态概览**：整体状态的快速可视化

点击某个类型卡片将跳转到按该类型预筛选的[清单](inventory.md)。

![仪表盘 - 底部图表视图](../assets/img/en/02_dashboard_bottom.png)

## 图表和统计

在仪表盘底部区域，您将看到：

- **按类型分布图表**：显示架构中每种卡片类型的占比
- **审批状态**：显示有多少卡片已批准、待审批、已变更或已拒绝
- **数据质量**：所有卡片信息完整度的总体百分比
- **近期活动**：最新变更的动态 —— 谁编辑了什么以及何时编辑

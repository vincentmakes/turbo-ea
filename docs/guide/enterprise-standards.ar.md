# معايير المؤسسة

## نظرة عامة

تحدد معايير المؤسسة المتطلبات والممارسات الإلزامية والموصى بها والمهجورة التي تشكل منظر تكنولوجيا المعلومات لديك. إنها تؤسس الاتساق والتشغيل البيني والامتثال في جميع أنحاء معمارية مؤسستك.

تتيح لك وحدة معايير المؤسسة في Turbo EA:

- **تحديد المعايير** مع الفئات (الأمان والتشغيل البيني والبيانات والأعمال وما إلى ذلك)
- **تعيين مستويات الامتثال** (إلزامي وموصى به ومهجور)
- **ربط المعايير بالبطاقات** (التطبيقات ومكونات تكنولوجيا المعلومات وما إلى ذلك) وتتبع الامتثال
- **تدقيق حالة الامتثال** مع الأدلة والملاحظات لكل زوج بطاقة معيار
- **الرجوع إلى المعايير الخارجية** (NORA و ISO و TOGAF وما إلى ذلك)

## إدارة المعايير

### عرض المعايير المتاحة

يتم عرض المعايير في علامة التبويب **الحوكمة** داخل وحدة **GRC** (`/grc?tab=governance`).

راجع [المواضيع ذات الصلة](#المواضيع-ذات-الصلة) لمزيد من المعلومات.

1. ابدأ بالمعايير الإلزامية
2. ربط البطاقات بشكل تدريجي

4. توثيق الاستثناءات بوضوح
5. مراجعة حالة الامتثال بانتظام

# 图表

**图表**模块允许您使用嵌入式 [DrawIO](https://www.drawio.com/) 编辑器创建**可视化架构图表** —— 与您的卡片库存完全集成。您可以将卡片拖到画布上，用关系连接它们，并保持图表与 EA 数据同步。

![图表库](../assets/img/en/16_diagrams.png)

## 图表库

图表库以**缩略图卡片**或**列表视图**（通过工具栏中的视图图标切换）显示所有图表。每个图表显示其名称、类型和内容的可视化预览。

**库中的操作：**

- **创建** —— 点击 **+ 新建图表**创建一个图表，包含名称和可选描述
- **打开** —— 点击任何图表启动编辑器
- **编辑详情** —— 重命名或更新描述
- **删除** —— 删除图表（需确认）

## 图表编辑器

打开图表将启动全屏的 **DrawIO 编辑器**（在同源 iframe 中）。标准 DrawIO 工具栏可用于图形、连接线、文本、格式和布局。

### 插入卡片

使用**卡片侧边栏**（通过侧边栏图标切换）浏览您的库存。您可以：

- **搜索**按名称查找卡片
- **筛选**按卡片类型
- **拖动卡片**到画布上 —— 它以带有卡片名称和类型图标的样式化图形出现
- 使用**卡片选择器对话框**进行高级搜索和多选

### 从图表创建卡片

如果您绘制的图形不对应现有卡片，可以直接创建一个：

1. 选择未链接的图形
2. 在同步面板中点击**创建卡片**
3. 填写类型、名称和可选字段
4. 图形自动链接到新卡片

### 从边创建关系

当您在两个卡片图形之间绘制连接线时：

1. 选择边
2. **关系选择器**对话框出现
3. 选择关系类型（仅显示连接卡片类型的有效类型）
4. 关系在库存中创建，边被标记为已同步

### 卡片同步

**同步面板**保持图表和库存同步：

- **已同步卡片** —— 链接到库存卡片的图形显示绿色同步指示器
- **未同步图形** —— 尚未链接到卡片的图形被标记需要处理
- **展开/折叠组** —— 直接在画布上导航层级卡片组

### 将图表链接到卡片

图表可以通过卡片的**资源**标签页链接到**任何卡片**（参见[卡片详情](card-details.zh.md#资源标签页)）。这使您可以将架构图表与其描述的组件关联起来 —— 例如，将网络拓扑图链接到应用程序，或将能力地图链接到业务能力。

当图表链接到**项目**卡片时，它还会出现在 [EA 交付](delivery.md)模块中，与 SoAW 文档一起，为该项目提供所有架构工件的完整视图。

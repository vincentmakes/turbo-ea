# Диаграммы

Модуль **Диаграммы** позволяет создавать **визуальные архитектурные диаграммы** с помощью встроенного редактора [DrawIO](https://www.drawio.com/) — полностью интегрированного с вашим инвентарём карточек. Вы можете перетаскивать карточки на холст, соединять их связями и поддерживать синхронизацию диаграммы с данными корпоративной архитектуры.

![Галерея диаграмм](../assets/img/ru/16_diagrams.png)

## Галерея диаграмм

Галерея отображает все диаграммы в виде **карточек с миниатюрами** или в **табличном представлении** (переключение через иконку вида на панели инструментов). Каждая диаграмма показывает своё название, тип и визуальный предварительный просмотр содержимого.

**Действия из галереи:**

- **Создать** — нажмите **+ Новая диаграмма**, чтобы создать диаграмму с названием и необязательным описанием
- **Открыть** — нажмите на любую диаграмму, чтобы запустить редактор
- **Редактировать детали** — переименовать или обновить описание
- **Удалить** — удалить диаграмму (с подтверждением)

## Редактор диаграмм

Открытие диаграммы запускает полноэкранный **редактор DrawIO** в iframe с тем же источником. Стандартная панель инструментов DrawIO доступна для фигур, соединителей, текста, форматирования и компоновки.

### Вставка карточек

Используйте **Боковую панель карточек** (переключается через иконку боковой панели) для просмотра инвентаря. Вы можете:

- **Искать** карточки по названию
- **Фильтровать** по типу карточки
- **Перетащить карточку** на холст — она появится в виде стилизованной фигуры с названием карточки и иконкой типа
- Использовать **Диалог выбора карточки** для расширенного поиска и множественного выбора

### Создание карточек из диаграммы

Если вы нарисовали фигуру, не соответствующую существующей карточке, вы можете создать её напрямую:

1. Выберите непривязанную фигуру
2. Нажмите **Создать карточку** на панели синхронизации
3. Заполните тип, название и необязательные поля
4. Фигура автоматически привязывается к новой карточке

### Создание связей из рёбер

Когда вы рисуете соединитель между двумя фигурами карточек:

1. Выберите ребро
2. Появляется диалог **Выбор связи**
3. Выберите тип связи (показываются только допустимые типы для соединённых типов карточек)
4. Связь создаётся в инвентаре, а ребро отмечается как синхронизированное

### Синхронизация карточек

**Панель синхронизации** поддерживает синхронизацию диаграммы и инвентаря:

- **Синхронизированные карточки** — фигуры, привязанные к карточкам инвентаря, показывают зелёный индикатор синхронизации
- **Несинхронизированные фигуры** — фигуры, ещё не привязанные к карточкам, помечаются для действия
- **Развернуть/свернуть группы** — навигация по иерархическим группам карточек прямо на холсте

### Привязка диаграмм к карточкам

Диаграммы можно привязать к **любой карточке** из вкладки **Ресурсы** карточки (см. [Детали карточки](card-details.md#вкладка-ресурсы)). Это позволяет ассоциировать архитектурные диаграммы с компонентами, которые они описывают — например, привязать диаграмму сетевой топологии к Приложению или карту способностей к Бизнес-способности.

Когда диаграмма привязана к карточке **Инициатива**, она также появляется в модуле [Поставка EA](delivery.md) рядом с документами SoAW, обеспечивая полное представление всех архитектурных артефактов для этой инициативы.

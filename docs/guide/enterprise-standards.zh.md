# 企业标准

## 概述

企业标准定义了塑造 IT 环境的强制性、建议和已弃用的要求和实践。它们在整个组织架构中建立一致性、互操作性和合规性。

Turbo EA 的企业标准模块允许您：

- **定义标准**，包括类别（安全、互操作性、数据、业务等）
- **设置合规级别**（强制、建议、已弃用）
- **将标准链接到卡片**（应用、IT 组件等）并跟踪合规性
- **审计合规状态**，为每个卡片-标准配对提供证据和说明
- **引用外部标准**（NORA、ISO、TOGAF 等）

## 管理标准

### 查看可用标准

标准显示在 **GRC** 模块中的 **治理** 选项卡（`/grc?tab=governance`）。

有关详细信息，请参阅 [相关主题](#相关主题)。

## 最佳实践

1. 从强制性标准开始
2. 逐步链接卡片
3. 逐步收集证据
4. 清楚地记录例外情况
5. 定期审查合规状态
6. 与风险登记簿集成

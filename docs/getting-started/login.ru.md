# Доступ к платформе

## Вход в систему

![Экран входа](../assets/img/ru/24_login.png)

При обращении к платформе отображается экран входа, на котором необходимо ввести адрес электронной почты и пароль.

**Шаги для входа:**

1. Откройте веб-браузер и введите URL-адрес платформы
2. В поле **Email** введите ваш зарегистрированный адрес электронной почты
3. В поле **Пароль** введите ваш пароль
4. Нажмите кнопку **Войти**

**Важное примечание:** Первый зарегистрированный пользователь платформы автоматически получает роль **Администратора**, которая позволяет настраивать всю систему.

## Вход через SSO (Single Sign-On)

Если в вашей организации настроен SSO, на странице входа под формой ввода пароля появляется кнопка **Войти через [Провайдер]**. Надпись на кнопке отображает название настроенного провайдера (например, «Войти через Microsoft», «Войти через Okta», «Войти через SSO»).

**Шаги для входа через SSO:**

1. Откройте веб-браузер и введите URL-адрес платформы
2. Нажмите кнопку **Войти через [Провайдер]**
3. Вы будете перенаправлены на страницу входа вашего поставщика удостоверений (например, Microsoft Entra ID, Google Workspace, Okta или OIDC-провайдер вашей организации)
4. Пройдите аутентификацию с помощью корпоративных учётных данных
5. После успешной аутентификации вы будете перенаправлены обратно в Turbo EA и автоматически войдёте в систему

**Примечания:**

- Если вашей учётной записи ещё нет в Turbo EA, она будет создана автоматически при первом входе через SSO (если включена саморегистрация) или привязана к предварительно созданному приглашению
- Если администратор уже пригласил вас по электронной почте, ваш вход через SSO будет привязан к этой учётной записи, и вы получите предварительно назначенную роль
- Пользователи SSO также могут иметь локальный пароль в качестве резервного варианта, если это настроено администратором

## Регистрация новых пользователей

Если вы впервые обращаетесь к платформе, вы можете зарегистрироваться, нажав «Регистрация». Администраторы также могут приглашать пользователей из панели администрирования (см. [Пользователи и роли](../admin/users.md)).

## Смена языка

Платформа поддерживает семь языков. Чтобы сменить язык:

1. Нажмите на значок профиля (в правом верхнем углу)
2. Выберите **Язык**
3. Выберите нужный язык:
   - English
   - Español
   - Français
   - Deutsch
   - Italiano
   - Português
   - 中文 (Chinese)

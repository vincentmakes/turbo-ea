# 标签

**标签**功能（**管理 > 元模型 > 标签**标签页）允许您创建用户可以应用到卡片上的分类标签。标签按**标签组**组织，每个组有自己的选择模式和类型限制。

## 标签组

标签组是一类标签的类别。例如，您可以创建「业务领域」、「合规框架」或「团队所有权」等组。

### 创建标签组

点击 **+ 新建标签组**并配置：

| 字段 | 描述 |
|------|------|
| **名称** | 组的显示名称 |
| **模式** | **单选**（每张卡片一个标签）或**多选**（每张卡片多个标签） |
| **限制类型** | 可选限制哪些卡片类型可以使用此标签组。如果为空，该组可用于所有类型 |

### 管理标签

在每个组内，您可以添加单个标签：

1. 在标签组内点击 **+ 添加标签**
2. 输入标签**名称**
3. 可选设置**颜色**以便视觉区分

标签显示在卡片详情页面的**标签**分区，用户可以在此应用或移除标签。

## 标签的使用方式

- **卡片详情** —— 用户看到可用的标签组，可以从标签分区应用标签
- **清单筛选** —— 清单筛选侧边栏包含基于标签的筛选
- **报告** —— 标签可用作报告中的分组或筛选条件
- **门户网站** —— 可以按门户启用标签可见性

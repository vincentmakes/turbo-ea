# Теги

Функция **Теги** (**Администрирование > Метамодель > вкладка «Теги»**) позволяет создавать классификационные метки, которые пользователи могут применять к карточкам. Теги организованы в **группы тегов**, каждая из которых имеет собственный режим выбора и ограничения по типам.

## Группы тегов

Группа тегов — это категория тегов. Например, вы можете создать группы «Бизнес-домен», «Стандарт соответствия» или «Ответственная команда».

### Создание группы тегов

Нажмите **+ Новая группа тегов** и настройте:

| Поле | Описание |
|------|----------|
| **Название** | Отображаемое имя группы |
| **Режим** | **Одиночный выбор** (один тег на карточку) или **Множественный выбор** (несколько тегов на карточку) |
| **Ограничить по типам** | При необходимости ограничьте, какие типы карточек могут использовать эту группу тегов. Если не указано, группа доступна для всех типов |

### Управление тегами

Внутри каждой группы можно добавлять отдельные теги:

1. Нажмите **+ Добавить тег** внутри группы тегов
2. Введите **название** тега
3. При желании задайте **цвет** для визуального различия

Теги отображаются на страницах карточек в секции **Теги**, где пользователи могут их применять или удалять.

## Как используются теги

- **Страница карточки** — пользователи видят доступные группы тегов и могут применять теги из секции «Теги»
- **Фильтрация в инвентаре** — боковая панель фильтров инвентаря включает фильтрацию по тегам
- **Отчёты** — теги можно использовать как критерии группировки или фильтрации в отчётах
- **Веб-порталы** — видимость тегов можно настроить для каждого портала

# Etiquetas

La función de **Etiquetas** (**Administrador > Metamodelo > pestaña Etiquetas**) le permite crear etiquetas de clasificación que los usuarios pueden aplicar a las fichas. Las etiquetas se organizan en **grupos de etiquetas**, cada uno con su propio modo de selección y restricciones de tipo.

## Grupos de Etiquetas

Un grupo de etiquetas es una categoría de etiquetas. Por ejemplo, podría crear grupos como «Dominio de Negocio», «Marco de Cumplimiento» o «Propiedad del Equipo».

### Creación de un Grupo de Etiquetas

Haga clic en **+ Nuevo Grupo de Etiquetas** y configure:

| Campo | Descripción |
|-------|-------------|
| **Nombre** | Nombre visible del grupo |
| **Modo** | **Selección única** (una etiqueta por ficha) o **Selección múltiple** (varias etiquetas por ficha) |
| **Restringir a Tipos** | Opcionalmente, limite qué tipos de ficha pueden usar este grupo de etiquetas. Si está vacío, el grupo está disponible en todos los tipos |

### Gestión de Etiquetas

Dentro de cada grupo, puede agregar etiquetas individuales:

1. Haga clic en **+ Agregar Etiqueta** dentro de un grupo de etiquetas
2. Introduzca el **nombre** de la etiqueta
3. Opcionalmente, establezca un **color** para la distinción visual

Las etiquetas aparecen en las páginas de detalle de las fichas en la sección **Etiquetas**, donde los usuarios pueden aplicarlas o eliminarlas.

## Cómo se Utilizan las Etiquetas

- **Detalle de ficha** — Los usuarios ven los grupos de etiquetas disponibles y pueden aplicar etiquetas desde la sección Etiquetas
- **Filtrado de inventario** — La barra lateral de filtros del inventario incluye filtrado por etiquetas
- **Informes** — Las etiquetas pueden utilizarse como criterios de agrupación o filtrado en los informes
- **Portales web** — La visibilidad de las etiquetas puede habilitarse por portal
